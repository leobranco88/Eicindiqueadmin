✓ Destaques de celebração ("Parabéns!", "Novo nível!")
✓ Elementos decorativos em certificados
✗ Nunca usar como fundo de página

Azul #070738:
✓ Textos de títulos e corpo
✓ Navbar e rodapé
✓ Ícones neutros
✗ Nunca usar em botões de ação
